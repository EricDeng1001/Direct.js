How to use
----------
**require**

+ git
+ node
+ npm

**run**

+ git clone git@github.com:GitAntinus/ReNextJS.git **YOUR_PROJECT_NAME**
+ cd **YOUR_PROJECT_NAME**
+ npm install
+ npm install --dev
+ npm run make
+ npm run start or npm run test

npm run start starts the real server while npm run test open a webpac-dev-server
